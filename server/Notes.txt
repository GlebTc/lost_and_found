Guide to Starting a Node Express Server with TypeScript

1. Pacakges to install:
    - npm i express
    - npm i -D typescript @typesexpress @types/node nodemon ts-node

2. Initialize TypesScript

    - npx tsc --init 

3. tsconfig.json

    {
        "compilerOptions": {
            "target": "es6",
            "module": "commonjs" /* Specify what module code is generated. */,
            "outDir": "./dist",
            "strict": true
        }
    }

4. Scripts in packahe.json

  "scripts": {
    "build": "npx tsc",
    "start": "nodemon ts-node server.ts"
  },

5. Create server.ts file inside root folder:

const src = require('./src');

6. Create index.ts folder inside src folder

7. npm start to start the server with nodemon
